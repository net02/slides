Data structures

A particular way of organizing data in a computer so that it can be used efficiently

Efficient data structures are key to designing efficient algorithms

Static vs Linked

Static Data structures

  • Array
  • Record
  • Union

Array

Contiene un numero di elementi (dello stesso tipo) in un determinato ordine.
Gli elementi sono individuati attraverso un indice numerico

Record (struct)

Contiene un numero di campi (dello stesso tipo) di tipo eterogeneo.
I campi sono individuati attraverso un nome

struct date {
   int year;
   int month;
   int day;
};

Union

Specifica due o più tipi di dato che può contenere.

Anche Tagged Union
